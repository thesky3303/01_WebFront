분기문

        - 반복문 수행 중 반복을 멈추거나 넘어가는 구문

        - break : (가장 가까운/감싸고 있는)반복문을 멈춤

        - continue : (가장 가까운/감싸고 있는)반복문의 시작 부분으로 이동
                    -> 다음 반복으로 넘어감

UP & DOWN GAME

    1부터 200 사이 난수를 몇 회 만에 맞추는지 카운트하는 게임

    //맞춰야되는 난수(1~200) 발생
    const answer = Math.floor( Math.random() * 200 ) + 1;

    - Math.floor()  : 내림 처리해서 정수로 변환
    - while()
    - break;
